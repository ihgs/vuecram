= 塾システムv2


== 利用システム

* firebase

== 開発

. firebase にプロジェクト作成
. `Webアプリにfirebaseを追加`　で認証情報を `src/conf.js` に保存
+
[source]
----
export default {
  apiKey: 'xxxxxxxxxxx',
  authDomain: 'xxxxxxxxx',
  databaseURL: 'xxxxxxxxxx',
  projectId: 'xxxxxxxxxx',
  storageBucket: 'xxxxxxxxx',
  messagingSenderId: 'xxxxxxxxxx'
}
----



=== How to use


firebase realtime db に以下のデータを作成

```
settings/tokens/(tokenid)/token=[value]
```

valueで設定した値を、curl でACCESS_TOKENで指定します
